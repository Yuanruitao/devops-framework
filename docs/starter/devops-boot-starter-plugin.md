# devops-boot-starter-plugin

`starter-plugin`组件帮助开发者实现插件机制，动态扩展系统功能

## 功能介绍
- 支持动态加载、卸载插件
- 支持通过`actuator`管理插件
- 支持扩展Point，动态添加逻辑
- 支持扩展Controller，动态增加接口


## 使用方式
- **build.gradle.kts**

```kotlin
implementation("com.tencent.devops:devops-boot-starter-plugin")
```

- **build.gradle**

```groovy
implementation 'com.tencent.devops:devops-boot-starter-plugin'
```
